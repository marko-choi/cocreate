CSV Configuration Test

This demonstrates how to use the new CSV configuration feature.

Usage Examples:

1. Without CSV configuration (default behavior):

loadReactApp(qualtricsSurveyEngine);

This will show only the functionality section (default behavior).

2. With CSV configuration URL:

loadReactApp(qualtricsSurveyEngine, "https://rotman.pdx1.qualtrics.com/ControlPanel/File.php?F=F_ABCD");

This will fetch the CSV configuration from the provided URL and show/hide sections accordingly.

3. Sample CSV format:

functionality,true
aesthetics,false
comments,true

4. Local testing with sample file:

loadReactApp(qualtricsSurveyEngine, "./sample-config.csv");

This will use the local sample configuration file for testing.

Configuration Options:

  • functionality: true/false - Controls whether the functionality thumbs up/down section is shown
  • aesthetics: true/false - Controls whether the aesthetics thumbs up/down section is shown
  • comments: true/false - Controls whether the additional comments textarea is shown

Default Configuration:

If no CSV URL is provided or if the CSV fetch fails, the default configuration is:

  • functionality: true
  • aesthetics: false
  • comments: false
